# ビュー（VIEW）（SQL）

Created: 2022年2月14日 18:33

テーブルの特定部分や複数テーブルを結合し、ユーザに必要な部分だけをあたかも一つのテーブルであるかのように見せるための、仮想的なテーブルの概念

SELECT文の結果を一覧として出せる

例）CREATE VIEW usersビュー AS SELECT ID, 名前 FROM usersテーブル;

とした場合、

SELECT ID, 名前 FROM usersテーブル;

と

SELECT * FROM usersビュー;

が同様となる。

ビューを利用することにより、実体のテーブルに格納されているデータへのアクセスを制限することが可能。

不用意にデータを更新してしまうトラブルを防ぐことが出来る。

実際にユーザーが必要なデータのみを抜粋したビューへのアクセスに制限することで、セキュリティを強化することにも繋がる。
